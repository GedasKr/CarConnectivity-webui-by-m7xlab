# Icons

This directory contains SVG icons for the CarConnectivity WebUI.

Icons are from Heroicons (https://heroicons.com/) - MIT Licensed

## Usage in Templates

```django
{% load static %}
<img src="{% static 'icons/car.svg' %}" alt="Car" class="icon">
```

## Icon List

- car.svg - Vehicle icon
- location.svg - Location/GPS pin
- battery.svg - Battery/charge indicator
- fuel.svg - Fuel indicator
- lock.svg - Lock status
- unlock.svg - Unlock status
- settings.svg - Settings/configuration
- user.svg - User profile
- logout.svg - Logout
- menu.svg - Hamburger menu
- close.svg - Close/X
- chevron-down.svg - Dropdown indicator
- refresh.svg - Refresh/reload
- info.svg - Information
- warning.svg - Warning
- check.svg - Success/checkmark
- x-circle.svg - Error/close
